jms_service

Version: jms_service_2-5-0

Id Type Summary Release Notes
JMSSERVICE-65 New Feature Support publishing messages to queue Fixed two problems with JMS ESA's JmsService.sendMessage() method 1. Method was waiting for a reply. 2. ESA did not allow to use Topics. To fix this problems the following changes have been implemented: 1. Made old JmsQueueOperation deprecated. 2. Provide a JmsQueuePublishOperation that publishes a message to a queue. 3. Provide a JmsQueueRequestOperation as replacement of JmsQueueOperation that implements request/reply on a queue.
JMSSERVICE-64 New Feature Support sending persistent messages JMS ESA now allows to configure the delivery mode of messages with the JmsInterfaceConfig boolean property 'persistentDelivery'. The default is NON_PERSISTENT mode.
